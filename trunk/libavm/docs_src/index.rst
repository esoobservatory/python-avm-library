Welcome
=======

*Python AVM Library* is a library for working with `Astronomy Visualization Metadata <http://www.virtualastronomy.org>`_
in the `XMP <http://www.adobe.com/products/xmp/>`_ format. 

**Authors:**
 * Amit Kapadia <akapadia@eso.org>
 * Lars Holm Nielsen <lnielsen@eso.org>
